## Overview

`IconTile` is a general-use tile to present data to the user. This can be
an info/error tile, or a stats tile.

## Usage

Currently, the icon only shows in `white`, so the `color` passed to the `IconTile`
must be dark enough to provide enough contrast and be accessible.

You may pass only a `message`
